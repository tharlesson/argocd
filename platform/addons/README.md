# Addons de Plataforma

Esta pasta documenta o limite de responsabilidade dos addons.

- Instalação de addons de infra (Helm + IRSA): `terraform/modules/addons`
- Configuração GitOps dos addons (manifests): `platform/*`

Addons cobertos:

- Argo CD
- Argo Rollouts
- kube-prometheus-stack
- External Secrets Operator
- AWS Load Balancer Controller
- Kyverno
